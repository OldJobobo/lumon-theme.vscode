# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

## [Unreleased]

## [0.0.6] - 2026-03-26

### Changed
- Refined the Marketplace README copy.

## [0.0.5] - 2026-03-25

### Changed
- Expanded syntax highlighting coverage to match the extended TextMate scope set used in the companion Miasma and Retro-82 themes.
- Added semantic token coverage and enabled semantic highlighting for stronger symbol separation across supported languages.
- Tuned publish-facing theme UI states for readable status bar, notification, and badge contrast.
- Added the Marketplace extension icon.

## [0.0.4] - 2026-03-25

### Changed
- Corrected the extension repository URL so the Marketplace README resolves `preview.png` from the VS Code theme repo.
- Rebuilt the packaged VSIX with the corrected Marketplace preview image source.

## [0.0.3] - 2026-03-25

### Changed
- Prepared the extension for a fresh Marketplace publish and rebuilt the packaged VSIX artifact.

## [0.0.2] - 2026-03-25

### Changed
- Renamed the bundled theme file to `lumon-color-theme.json`.
- Updated extension metadata and publishing docs to reference the new theme path.

## [0.0.1] - 2026-03-25

### Added
- Initial Lumon VS Code theme extension release.
- Cold blue Lumon palette mapped across VS Code UI, syntax tokens, and terminal ANSI colors.
